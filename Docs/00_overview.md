## シャドウマップとは何か

### 通常の描画（1パス）

```
カメラ → シーン → 画面に色を出す
```

これだけでは「影」は生まれません。  
カメラには「他のオブジェクトに遮られているか」がわからないからです。

### シャドウマップ（2パス）

```
★ パス1（シャドウパス）
   ライト → シーン → 深度テクスチャに「ライトからの距離」だけを書き込む

★ パス2（メインパス）
   カメラ → シーン → 「このピクセルはライトから見えていたか？」を
                      パス1の深度テクスチャと照合して判定
                      → 見えていた = 明るい
                      → 見えていなかった = 影
```

イメージとしては**ライトの目線で写真を撮る**ようなものです。  
その写真（＝深度テクスチャ）と、実際の描画を比べて影を判定します。

## 現在のプロジェクト構成（開始時点）

```
Simple3D.hlsl       ← メインの3Dシェーダー（ここに影判定を追加する）
Stage.cpp / Stage.h ← シーン管理・コンスタントバッファ送信
Direct3D.cpp / .h   ← デバイス・シェーダー初期化
Engine/Model.cpp    ← モデル描画
```

```hlsl
float4 lightPosition;  // 平行光源の方向ベクトル（または点光源の位置）
float4 eyePosition;    // カメラ位置
int    lightType;      // 0=平行光源, 1=点光源
float3 _pad;           // アライメント用
```

Step5 でここに `matLightVP`（ライト視点のVP行列）を追加します。
